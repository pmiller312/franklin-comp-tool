HOW TO SELECT COMPS
1. Fetch data using the query above
2. Click ☑ Select Comps button (appears above results list)
3. Click dots on the map or items in the list to select up to 5 sales
4. Click ⬇ Generate Sales Sheet to download a DOCX comp data sheet
✓ VACANCY VERIFICATION
All results are filtered to Ohio DTE vacant land class codes (100, 110, 300, 400, 500–505) and pass a dual improvement test on every parcel: ① BLDVALUEBASE = 0 and ② TOTVALUEBASE = LNDVALUEBASE (no improvement component in assessed value). Optionally filter further by Max Improvement Value to allow a small tolerance.
For multi-parcel sales, all parcels in the same deed (matched by date + price + owner) must pass — if any parcel carries building value, the entire sale is excluded.
🛰 Imagery: Button cycles Dark → OSIP Ohio (6-inch, public domain, © OGRIP/OIT) → Esri World Imagery. OSIP is Ohio-specific leaf-off orthoimagery; resolution and vintage vary by county.

⚠ Razed-site detection is heuristic. Signals: keyword match in legal/deed (strongest, +3), utilities on vacant parcel (+1/+2), sale price 20×+ assessed land (+1/+2), platted lot number (+1). Score ≥ 2 = flagged. Note: Ohio assesses at ~35% of market so normal vacant land already trades at 5-6× assessed — only extreme outliers (20×+) are scored. Both improvement tests reflect assessed values, not physical inspections. A recently demolished structure or assessment lag could cause either field to lag behind reality. Independent verification is recommended for individual comps.
